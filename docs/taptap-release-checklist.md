# TapTap 上架检查清单

## 包体

- [ ] signed APK 已生成
- [ ] 包名与 `tap-game.config.json` 一致
- [ ] `versionCode` / `versionName` 正确
- [ ] APK 可安装
- [ ] 可完整完成一局或一个核心关卡
- [ ] 没有依赖本地 `localhost`

## 素材

- [ ] `artifacts/store-assets/icon-1024.png` 存在，尺寸为 1024 x 1024
- [ ] 至少 3 张真实游戏截图
- [ ] `artifacts/store-assets/promo-16x9.png` 存在，比例为 16:9
- [ ] 宣传图只放审核允许的文字，优先只保留游戏名
- [ ] `artifacts/store-assets/video/gameplay.mp4` 存在
- [ ] 实机视频不少于 15 秒，建议 20-60 秒
- [ ] 实机视频前 5 秒内展示核心玩法
- [ ] 核心玩法展示至少持续 10 秒
- [ ] 视频无额外人声配音
- [ ] 视频无字幕
- [ ] 视频无画中画
- [ ] 视频不露出手机设备 UI 或浏览器地址栏
- [ ] 视频不展示折扣充值等纯商业内容
- [ ] 视频编码为 H.264 或 HEVC/H.265
- [ ] 视频短边不低于 540 px
- [ ] 视频文件小于 5GB

## 文案

- [ ] 游戏一句话简介已填写
- [ ] 游戏简介已填写
- [ ] 核心玩法说明已填写
- [ ] 版本说明已填写
- [ ] AI 内容说明已填写
- [ ] 年龄分级说明已填写

## 隐私与合规

- [ ] `privacy.html` 已生成
- [ ] 公网隐私政策链接可访问
- [ ] 隐私政策与游戏实际行为一致
- [ ] 未上线在线 AI 时，明确写“不提供在线 AI 合成服务”
- [ ] 无账号、无支付、无广告时，明确写清楚
- [ ] 如果未接入 TapTap 防沉迷 SDK，优先选择 TapPlay 分发

## 发布前最终复核

- [ ] `npm run taptap:check` 通过
- [ ] 上传前重新打开 APK 测试一次
- [ ] TapTap 后台填写的游戏名、包名、版本号、隐私链接一致
